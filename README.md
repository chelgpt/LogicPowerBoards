# LogicPowerBoards

KiCad/KiCanvas workspace for the Docreate 002 modernization boards.

This repository is prepared for browser-based review through GitHub Pages + KiCanvas.

## Boards

- PowerBoard: 24V input, 5V / 20A isolated DC-DC board based on Mornsun URF2405QB-100WR3.
- LogicBoard: 24V input, 12V / 6W isolated DC-DC board based on Mornsun URB2412ZP-6WR3.
